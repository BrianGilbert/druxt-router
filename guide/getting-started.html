# Getting started

# Druxt.js

This module is included in the core Druxt.js project, the below instructions are only required when installing this module standalone.

# Quickstart

  1. Install the NPM module:

    npm i druxt-router
    1

  2. Add the module and configure as required in nuxt.config.js:

    module.exports = {
  modules: [
    'druxt-router'
  ],
  druxt: {
    baseUrl: 'https://example.com'
  }
}

# Requirements

  • Drupal JSON:API backend.
  • Decoupled router Drupal module.

# Drupal Druxt.js module

This module was designed for Druxt.js and work best with the the Drupal Druxt.js module.
