# druxtRouterget(path) ⇒ object

Get route and redirect information. - Dispatches `druxtRouter/getRoute` action. - Sets the active route. - Sets the active redirect.
Vuex Action => Mutates state propertyroute

Kind: action method of druxtRouter
Returns: object - The route and redirect information.

Param Type Description
path string The router path.

Example

const { redirect, route } = await this.$store.dispatch('druxtRouter/get', '/')
1

# druxtRoutergetEntity(query) ⇒ object

Get Entity. - Executes query against Drupal JSON:API. - Caches result in the Vuex store. - Returns cached result from Vuex store when available.
Vuex Action => Mutates state propertyentities

Kind: action method of druxtRouter
Returns: object - The Drupal entity JSON:API resource data.

# druxtRoutergetResources(context) ⇒ Array.<object>

Get multiple resources.
Vuex Action => Mutates state propertyundefined

Kind: action method of druxtRouter
Returns: Array.<object> - Array of Drupal JSON:API resource data.
Todo

  • Add Vuex store caching for getResources.
Param Type Default Description
context object Object containing druxtRouter.getResources() parameters.
context.resource string The JSON:API resource type.
context.query string | object A JSON:API query string or object.
[context.options] object
[context.options.all] boolean false Load all results.

Example

// Load all currently published Articles.
const query = new DrupalJsonApiParams()
query.addFilter('status', '1')
const resources = await this.$store.dispatch('druxtRouter/getResources', {
  resource: 'node--article',
  query,
  options: { all: true }
})

# druxtRoutergetRoute(path) ⇒ object

Get Route. - Executes query against Drupal Decoupled router. - Caches result in the Vuex store. - Returns cached result from Vuex store when available.
Vuex Action => Mutates state propertyroute

Kind: action method of druxtRouter
Returns: object - The route object.

Param Type Description
path string The route path.

Example

const route = await this.$store.dispatch('druxtRouter/getRoute', '/')
